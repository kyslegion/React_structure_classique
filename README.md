# La structuration de base d'un projet Vite 
```
my-vite-project/
  ├── public/
  │   ├── index.html
  │   └── ...
  ├── src/
  │   ├── main.js
  │   ├── App.vue
  │   └── ...
  ├── assets/
  │   ├── images/
  │   ├── styles/
  │   └── ...
  ├── tests/
  ├── package.json
  ├── README.md
  └── ...
Note: index.html n'est lu que dans le dossier général et pas dans public

```
